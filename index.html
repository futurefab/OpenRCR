What is GitHub?

We have a guide explaining this here. But briefly, projects like OpenRCR can get very complicated. There are scores of parts, files and documents that are all regularly updated. To keep control of these versions is a lot of work. GitHub is a well established web service that manages all this very easily. All you need to do is worry about downloading the latest compressed .ZIP file and working through the tutorials to make your home made R/C Car.

This is all free?

It sure is! All the files you need will be free and always will be but if you want to order a kit then there will be a small charge to cover manufacturing costs. If you want to register a team for competition, there is a small admin charge to cover event costs but other than that,

I'd like to donate.

Excellent! FutureFab is a small organisation and the OpenRCR competition needs funds to run web servers, cover insurance and venue costs, so if you would like to support us, then you are most welcome and donations of any size will be greatly received! [Donation Link to Follow]

What do I need to make my OpenRCR Car?

The short answer is some hand tools, a soldering iron, access to a 3D Printer and some hobbyist radio gear. The long answer is documented in the blog and tutorials. Check out our get started guide to get the low down on your next steps. If you want to purchase a kit, to speed things up a bit you can by visiting our shop.

What is OpenSource all about?

We have a full guide on this which can be found here. In short, it's a distribution model where not only is the end product freely available but also the files used to make that product. This concept started with software where developers would share their code for others to edit. The OpenRCR project uses the well established OpenHardware principle. All of the car's files are free to download, edit and redistribute as you seen fit.
